# discord-bot-starter

**For production use**

`npm start`

**Development**

`npm run dev`

**Reconfigure slash(/) commands**

`npm run commands`

**Deletes all guild slash(/) commands**

`npm run delete`

**Lint all files**

`npm run lint`

**Update node modules to latest minor release**

`npm run update`

**Force upgrade all node modules to latest release**

`npm run upgrade`
